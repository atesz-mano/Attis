# Pasi a Háznál - Ezermester Szolgáltatások

Egy professzionális weboldal egy megbízható ezermester vállalkozás számára.

## 📋 Tartalom

- **index.html** - Főoldal
- **css/style.css** - Teljes stílus és responsív design
- **js/script.js** - Interaktív funkciók
- **php/sendmail.php** - E-mail küldés (sablonként, aktiválásra vár)

## 🎨 Jellemzők

✅ Reszponzív design (mobil, tablet, asztali)  
✅ Gyors és optimalizált  
✅ Könnyen testreszabható  
✅ Modern, professzionális megjelenés  
✅ Ajánlatkérő form  
✅ Sima görgetés navigáció  
✅ Animációk és vizuális effektek  

## 🚀 Szolgáltatások

- Konyhai és otthoni javítások
- Bútorösszeszerelés
- Csaptelepcsere
- Szilikonozás

## 📧 Kapcsolat

Az ajánlatkérő form jelenleg demo módban működik. Az e-mail küldés bekapcsolásához a `php/sendmail.php` fájlt kell konfigurálni.

## 🎯 Következő lépések

1. Domain regisztráció
2. Webhosting bérlése
3. E-mail küldés beépítése (sendmail.php konfigurálása)
4. Google Analytics hozzáadása
5. SEO optimalizáció
